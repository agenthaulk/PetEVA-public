# EVA Unit-02 Pet Assets

This folder contains the public Unit-02 asset pack used by PetEVA.

Included files:

- `pet.json` - Codex custom pet manifest.
- `spritesheet.webp` - Codex-compatible `1536 x 1872` atlas.
- `source/frames/unit02-*.png` - transparent frame sources used by the macOS runtime.
- `source/frames/2026-05-13-unit02-motion-12frame/runtime-12frame-clean/` - twelve-frame movement loops.

The assets are fan-made/generated and are not official Evangelion production art.
